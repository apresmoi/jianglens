The page teaches a usable mechanism rather than summarizing one lecture. The reader can apply the lens by asking what code selects an administrative class, what loyalty problem the code solves, what rival loyalties it defeats, and whether the filter still serves the institution's living purpose.

The strongest boundary is that bureaucracy names the institutional form, while this page names the code that makes officeholders selectable and loyal.

Residual reader risk: the title still sounds broad. The body mitigates this by repeatedly returning to administrators, officeholding, staffing, loyalty, status access, ritual, language, and method. Future additions should not expand it into a general theory of religion or secular ideology.
