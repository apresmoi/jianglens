# Lens

`lens/` holds the reconstructed Jiang world model and related layers.

- `canon/` is the reviewed Jiang-source-grounded reconstruction.
- `glossary/` stores atomic terms and concepts.
- `ledger/` stores Jiang's diagnoses and predictions.
- `evidence/` links claims to source spans.
- `commentary/` stores external support, challenge, reception, and context.
- `runtime/` stores skills or prompt packs compiled from the canon.

The canon is not a style imitation layer and not a place for unsupported synthesis.
