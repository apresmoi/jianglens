# Plato Proposals

Use this folder for process or skill proposals that Plato should not apply directly.

Good proposals:

- new lens seed clustering methodology,
- validation scripts for lens points or episode-to-lens links,
- public concept page structure improvements,
- corpus impact schema changes,
- runtime changes for a future Plato durable loop.

Do not edit `.codex/skills/**` directly. Propose changes here or in PR notes.
