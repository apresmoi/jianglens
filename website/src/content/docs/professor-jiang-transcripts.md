# Professor Jiang Transcripts

Jiang Lens turns public Predictive History videos into episode pages with compact readings, timestamped source trails, and full transcript views. The transcript is the archive. The episode reading is the compressed public path through it.

The goal is not to publish raw ASR as the main experience. Lectures repeat, circle, restart, and contain noisy transcription. The public page should preserve Jiang's strongest argument in readable form while keeping the transcript available for verification.

## What Each Episode Should Expose

An episode should show:

- the video and source date,
- a compact reading in Jiang's conceptual voice,
- timestamped links from prose back to the video,
- student questions only when they actually appear and Jiang answers them,
- full transcript access for audit,
- generated data for agents.

## Machine-Readable Transcript Links

Agents should use [`/llms.txt`](/llms.txt), [`/llms-full.txt`](/llms-full.txt), [`/data/lens/manifest.json`](/data/lens/manifest.json), and [`/data/lens/link-index.json`](/data/lens/link-index.json) to find the latest published transcript routes and source refs.
